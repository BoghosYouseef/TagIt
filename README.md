# TagIt

## Overview
TagIt is a desktop application designed to provide tagging functionality for various items. Built with Spring Boot and JavaFX, it offers a modern graphical user interface for desktop environments.

## Installation

### Windows
1. Download the `TagIt-installer.exe` from the [Releases](https://github.com/your-username/TagIt/releases) page
2. Run the installer
3. Follow the installation wizard
4. TagIt will be installed and a desktop shortcut will be created
5. Launch TagIt from your Start Menu or desktop shortcut

### macOS
1. Download the `TagIt.dmg` from the [Releases](https://github.com/your-username/TagIt/releases) page
2. Open the DMG file
3. Drag TagIt to your Applications folder
4. Launch TagIt from your Applications folder

### Linux
1. Download the `tagit.deb` or `tagit.rpm` from the [Releases](https://github.com/your-username/TagIt/releases) page
2. Install using your package manager:
   ```
   # For Debian/Ubuntu
   sudo dpkg -i tagit.deb
   
   # For RedHat/Fedora
   sudo rpm -i tagit.rpm
   ```
3. Launch TagIt from your applications menu

## Features
- Tag management interface with tabbed navigation
- Spring Boot backend with Spring configuration
- Cross-platform support (Windows, macOS, Linux)
- Lightweight and fast performance
